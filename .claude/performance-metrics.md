# Performance Metrics

## CRUD 속도

[PM-01] Create 속도: 데이터 생성 시간 측정 (ms)
[PM-02] Read 속도: 데이터 조회 시간 측정 (ms)
[PM-03] Update 속도: 데이터 수정 시간 측정 (ms)
[PM-04] Delete 속도: 데이터 삭제 시간 측정 (ms)

## 검색 성능

[PM-05] 색인 검색: 인덱스가 적용된 필드 기준 검색 시간 측정 (ms)
[PM-06] 비색인 검색: 인덱스가 없는 필드 기준 검색 시간 측정 (ms)
[PM-07] 검색 결과: 검색된 데이터 개수 및 소요 시간 기록

## 리소스 사용량

[PM-08] 메모리 사용량: 테스트 수행 중 메모리 사용량 측정 (MB)
[PM-09] CPU 사용률: 테스트 수행 중 CPU 사용률 측정 (%)
[PM-10] 디스크 I/O: 읽기/쓰기 작업량 측정 (MB)

## 배치 작업

[PM-11] 배치 작업 처리량: 다수 데이터 일괄 처리 시간 측정 (ms)
[PM-12] 배치 크기: 한 번에 처리한 데이터 개수 기록

## 트랜잭션

[PM-13] 트랜잭션 처리 속도: 트랜잭션 내 복수 작업 수행 시간 측정 (ms)
[PM-14] 트랜잭션 작업 수: 트랜잭션 내 포함된 작업 개수 기록

## 초기화

[PM-15] 초기 로딩 시간: DB 초기화 및 첫 접근 소요 시간 측정 (ms)
[PM-16] DB 파일 크기: 초기화 후 생성된 DB 파일 크기 측정 (MB)

## 동시성

[PM-17] 동시성 처리: 동시 다중 작업 수행 시간 측정 (ms)
[PM-18] 동시 작업 수: 동시에 실행한 작업 개수 기록
[PM-19] 동시성 성공률: 정상 완료된 작업 비율 기록 (%)

## 쿼리 복잡도

[PM-20] 단순 조회: 단일 조건 검색 시간 측정 (ms)
[PM-21] 복합 조회: 다중 조건 검색 시간 측정 (ms)
[PM-22] 집계 쿼리: COUNT, SUM, AVG 등 집계 연산 시간 측정 (ms)
[PM-23] 정렬 쿼리: ORDER BY 적용 시 시간 측정 (ms)
