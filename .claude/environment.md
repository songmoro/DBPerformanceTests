# Environment

## 하드웨어 정보

[ENV-01] CPU 모델: 프로세서 모델명 기록
[ENV-02] CPU 코어 수: 물리 코어 및 논리 코어 수 기록
[ENV-03] RAM 크기: 총 메모리 용량 기록 (GB)
[ENV-04] 디스크 타입: SSD/HDD 종류 및 모델 기록
[ENV-05] 디스크 용량: 총 용량 및 사용 가능 공간 기록 (GB)

## 소프트웨어 정보

[ENV-06] macOS 버전: 운영체제 버전 및 빌드 번호 기록
[ENV-07] Swift 버전: 컴파일러 버전 기록
[ENV-08] Xcode 버전: 개발 도구 버전 기록
[ENV-09] 데이터베이스 버전: 각 DB 프레임워크/라이브러리 버전 기록

## 시스템 리소스 상태

[ENV-10] CPU 사용률: 테스트 시작 전 시스템 CPU 사용률 측정 (%)
[ENV-11] 메모리 사용률: 테스트 시작 전 시스템 메모리 사용률 측정 (%)
[ENV-12] 메모리 압박 수준: 메모리 압박 상태 기록 (normal/warning/critical)
[ENV-13] 디스크 사용률: 테스트 시작 전 디스크 사용률 측정 (%)
[ENV-14] 디스크 I/O 대기: 디스크 I/O 대기 시간 기록

## 백그라운드 프로세스

[ENV-15] 실행 중인 프로세스 수: 전체 프로세스 개수 기록
[ENV-16] 주요 백그라운드 앱: CPU/메모리를 많이 사용하는 상위 5개 프로세스 기록
[ENV-17] 네트워크 활동: 네트워크 사용 중인 프로세스 기록

## 측정 시점

[ENV-18] 테스트 시작 전: 환경 정보 수집 및 기록
[ENV-19] 테스트 종료 후: 최종 리소스 상태 기록
[ENV-20] 테스트 일시: ISO 8601 형식으로 테스트 수행 시각 기록

## 환경 정리 지침

[ENV-21] 불필요한 백그라운드 앱 종료 권장
[ENV-22] 시스템 재시작 후 테스트 수행 권장
[ENV-23] 네트워크 활동 최소화 권장
[ENV-24] 화면 보호기 및 절전 모드 비활성화 권장
